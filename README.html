BaseCreate

Create-first • SMP-ready • Heavy pack

A Create-focused SMP modpack built for big factories, player-driven economy, and recursive automation. Packed with Create addons + performance/stability tweaks for long-term worlds.

Version: 1.0.0 • Minecraft: 1.20.1 (Forge) • Mods: 400+

What this pack is about

Economy through engineering

Progress comes from what you build. Specialize, trade, and scale production. If your system breaks, that’s part of the game.

Recursive automation

Multi-step production chains, logistics, trains, and factories that feed other factories. The fun is in improving the loop.

Create-only focus

Everything here exists to support Create gameplay (and keep it stable). No random tech trees, no magic bloat.

Server-first pack. This repo contains the pack’s original content (configs/scripts/docs). Mods remain the property of their authors.

Install (players)

Recommended: Prism Launcher / MultiMC import

Fastest way to get the exact same setup as the server.

  • Download the pack ZIP from the project releases.
  • Open Prism/MultiMC → Add Instance → Import.
  • Select the ZIP and launch.

Requirements

  • MC: 1.20.1
  • Loader: Forge
  • RAM: 6–10GB recommended

Tip: if you have 16GB+ system RAM, allocate ~8–10GB to the game.

One-click installs are messy for big server packs. Some launchers/platforms can’t legally bundle every mod. Importing the ZIP keeps versions consistent and avoids missing mods.

Server setup (admins)

Server files

Use the provided server pack (or sync configs/scripts from this repo). Keep server + client in lockstep to avoid desync issues.

Performance notes

The pack includes optimizations, but mixin-heavy mods can conflict. If you change performance mods, test on a fresh world first.
